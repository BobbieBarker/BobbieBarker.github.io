Bordered uppercase topic label for article tags and tag-filter lists; pass `active` for the selected filter and `count` to show post counts.

```jsx
<Tag>elixir</Tag>
<Tag as="a" href="/tag/llm" count={12}>llm</Tag>
<Tag active>dist-sys</Tag>
```

Square 2px corners, hairline border that brightens on hover when interactive. Use `size="sm"` inside dense post rows.
